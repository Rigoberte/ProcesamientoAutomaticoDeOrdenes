#### Descripción

**Tracking Number Generator** es una herramienta diseñada para automatizar el proceso de gestión y seguimiento de órdenes de envío en estudios clínicos. Esta aplicación simplifica el flujo de trabajo al permitir que los distintos equipos ingresen órdenes y las procesen de manera eficiente y precisa, reduciendo la posibilidad de errores humanos.

#### Problema

El proceso manual de gestión de órdenes de envío implica varios pasos que pueden ser propensos a errores y consumir mucho tiempo:
1. Cargar una tabla de Excel con las órdenes.
2. Cargar una tabla de contactos y los horarios de entrega de cada centro médico.
3. Recopilar manualmente la información de cada orden.
4. Ingresar manualmente los datos en la página del transportista para generar un número de seguimiento (Tracking Number).

#### Solución

**Tracking Number Generator** automatiza estos pasos, normalizando una tabla de Excel con los datos de las órdenes y cargando las órdenes en las páginas web de los transportistas preconfigurados. Esto permite adaptarse a diferentes tablas de órdenes y a distintas páginas web, garantizando un proceso más fluido y menos propenso a errores.

#### Funcionalidades

1. **Normalización de Datos**: La aplicación normaliza los datos de las órdenes provenientes de diversas tablas de Excel.
2. **Carga Automática en Páginas de Transportistas**: Las órdenes se cargan automáticamente en las páginas web de los transportistas preconfigurados.
3. **Generación de Tracking Numbers**: Se generan números de seguimiento (Tracking Numbers) automáticamente, facilitando el rastreo de las órdenes.
4. **Notificaciones**: Una vez generado el Tracking Number, se notifica al centro médico y, en algunos casos, al cliente, sobre la fecha de entrega de la orden.

#### Contexto

Cada cliente cuenta con estudios de ensayos clínicos y colabora con múltiples hospitales a nivel nacional. A cada hospital se le asigna un número de centro (o site#) para distinguirlo en el contexto de un ensayo clínico específico. Los hospitales, ubicados en distintas ciudades, tienen diferentes tiempos de tránsito y múltiples entradas y direcciones que deben estar debidamente registradas para garantizar la entrega correcta.

Cada estudio cuenta con personal designado en cada hospital, autorizado para recibir las órdenes de envío. Estas personas tienen días fijos de presencia y horarios específicos, además de sus propios teléfonos y correos electrónicos de contacto.

Las órdenes de envío están asociadas a un estudio específico y a un site identificado con un número IVRS, e indican una condición de temperatura y una fecha de solicitud. Nuestros clientes solicitan que junto con las órdenes de envío se impriman y envíen ciertos documentos siempre idénticos.

#### Beneficios

- **Eficiencia**: Reducción del tiempo y esfuerzo requerido para procesar las órdenes.
- **Precisión**: Minimización de errores humanos en la gestión de datos y generación de números de seguimiento.
- **Flexibilidad**: Adaptabilidad a diferentes tablas de órdenes y páginas web de transportistas.
- **Notificaciones Automatizadas**: Información precisa y oportuna sobre el estado de las órdenes para los centros médicos y clientes.

#### Conclusión

**Tracking Number Generator** es una solución integral para la automatización del proceso de gestión de órdenes de envío en estudios clínicos, optimizando el flujo de trabajo y asegurando la precisión y eficiencia en cada etapa del proceso.
